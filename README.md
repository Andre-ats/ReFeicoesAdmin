# Re_feições Admin Frontend

Frontend da **área administrativa** da plataforma **Re_feições**, um sistema de e-commerce voltado para a venda de refeições.

Esta aplicação é responsável pela interface utilizada pelos administradores da plataforma, permitindo o gerenciamento do catálogo, acompanhamento de pedidos e visualização de dados do sistema.

## 🚀 Tecnologias

* React
* TypeScript
* TailwindCSS
* Redux

## 📚 Sobre o Projeto

O **Re_feições** é uma plataforma de venda de marmitas congeladas online.
Este repositório contém o **frontend da área administrativa**, desenvolvido para que administradores possam gerenciar e acompanhar o funcionamento do sistema.

A aplicação foi construída utilizando **React com TypeScript**, garantindo maior organização e segurança no desenvolvimento, além de **TailwindCSS** para estilização da interface e **Redux** para gerenciamento de estado da aplicação.

## ⚙️ Funcionalidades

* Autenticação de administradores
* Cadastro de novos produtos
* Edição e gerenciamento de produtos
* Visualização e controle de pedidos
* Acompanhamento de vendas
* Visualização de estatísticas do sistema
